# 个人知识网站 UI 全面焕新设计

## 概述

对 mysite 个人知识网站进行全面的 UI/UX 改造，将原有的扁平设计升级为玻璃拟态（Glassmorphism）+ 暖色渐变风格，增加全屏 landing page、导航过渡动画、分色板块等特性。

## 设计目标

- **氛围**：温暖个人（Warm Personal）
- **主色调**：琥珀橙 `#d97706`（保留现有 accent 色）
- **技术路线**：增强 SSR（Option B）— 后端渲染 + JS 前端增强

## 页面设计

### 1. 首页 Landing Page

- **背景**：全屏风景图或暖色渐变（RGB 255,217,210 → RGB 210,255,251）
- **星星效果**：15+ 闪烁星光点缀，CSS animation 定时闪烁
- **品牌文字**：`SunHongxun'S Site` — Georgia 衬线斜体，1.8rem，深棕色，居中对齐
- **副标题**：`计算生物学 · 药物发现` — 浅棕色，letter-spacing 0.18em
- **分隔线**：32px 宽，1px 高，浅棕半透明
- **滚轮引导**：底部滚动指示器，闪烁动画
- **剪影**：底部山形曲线，半透明渐变覆盖
- **入场动画**：品牌文字 + 副标题 + 引导 依次 fade-up 入场（0.3s / 0.5s / 0.6s / 1s）

### 2. 导航栏

- **位置**：顶部居中
- **样式**：玻璃毛玻璃 — `backdrop-filter: blur(24px)`，半透明底色
- **左上角**：⛰️ emoji
- **中间**：6 个导航项（首页、关于我、研究、实验手册、抽屉、随笔）
- **右上角**：🌙/☀️ 主题切换按钮
- **活跃按钮**：使用对应板块颜色高亮
- **导航底色**：RGB(113, 187, 234) 低透明度作为玻璃氛围

### 3. 板块配色（暖→冷渐变）

| 板块 | 颜色 | 色值 |
|------|------|------|
| 01 关于我 | 温暖粉红 | RGB(254, 141, 159) |
| 02 我的研究 | 浅粉 | RGB(247, 226, 231) |
| 03 实验手册 | 极浅粉 | RGB(255, 225, 225) |
| 04 抽屉 | 极浅蓝 | RGB(227, 244, 255) |
| 05 随笔 | 浅蓝 | RGB(188, 227, 245) |
| 导航栏底色 | 蓝色 | RGB(113, 187, 234) |

### 4. 过渡动画

- **首页→内容**：旧内容 scale(1→0.9) + opacity(1→0) 0.3s → fetch 新内容 → 新内容 scale(0.85→1) + opacity(0→1) 0.4s
- **内容→首页**：内容区 scale(1→0.9) + opacity(1→0) 0.3s → landing 页 scale(0.85→1) + opacity(0→1) 0.4s
- **内容→内容**：同上，旧内容缩出 → 新内容放大

### 5. 内容页布局

- **背景**：暖白 `#f8f5f0`
- **文章卡片**：毛玻璃效果，使用对应板块颜色的 10-15% 透明度 + backdrop-blur，圆角 16px
- **文章头部**：板块标签（带颜色圆点）+ 标题 + 渐变分隔线
- **留言板**：与卡片风格统一
- **页脚**：简约居中，`© 2026 鸿勋 · 以 Markdown 书写`

### 6. 深色模式

- 保留现有深色切换逻辑
- 深色下毛玻璃效果使用深色半透明底色
- 板块颜色在深色模式下调亮
